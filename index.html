Mapping Hub

This is an experimental pre-release site for sharing data mappings that translate between data formats, data models and/or vocabularies. It is based on the Yosemite Project vision for improving healthcare data interoperability.

Mappings

The Mappings repository contains descriptions of mappings (a/k/a translators) that are used to convert source data formats, data models and/or vocabularies to target data formats, data models and/or vocabularies.

Visit Repo »

Elements

The Elements repository contains descriptions of source, target and/or intermediate data elements used by the mappings.

Visit Repo »

Views

The Views repository contains definitions of various auto-generated views of mappings and elements, enabling them to be organized and displayed in different ways.
